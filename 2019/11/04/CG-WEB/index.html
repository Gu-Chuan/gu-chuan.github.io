口令错误？F12查看页面信息。

口令长度为11，而输入的长度为10。

解题方式：

  1. maxlength=”10” 字段修改为 maxlength=”11” 或是更大的值 ，并输入zhimakaimen。

  2. post传值text1=zhimakaimen。
