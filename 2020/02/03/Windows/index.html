Windows

Feb 03 2020

浅记

Windows是美国微软公司( Microsoft )研发的一套操作系统，问世于1985年，起初是用作Microsoft-DOS模拟环境，而后续的发展使得其成为当前应用最广泛的操作系统。

Windows采用了图形化模式GUI，比起从前的Dos需要输入指令使用的方式，更为人性化。其具有支持多种应用软件、硬件的特点。此外，Windows 具有许多版本。

DOS 是磁盘操作系统 (Disk Operating System) 的缩写，是个人计算机上的一类单用户单任务的操作系统。DOS作为微软公司在个人计算机上使用的一个操作系统载体( 管理各种软硬件资源 )，Windows NT 问世以来，DOS就是一个后台程序的形式出现，可以通过点击运行 -CMD 进入运行。

CMD ( command ) 一种命令提示符，即提示进行命令输入的一种工作提示符。在不同的操作系统环境下，命令提示符各不相同。在windows环境下，命令行程序为cmd.exe，是一个32位的命令行程序( Windows上的命令解释程序 )。其是Windows环境下的 “假DOS”，支持和使用部分DOS命令。

Windows的CMD命令基于DOS命令而发展形成的。DOS命令可以删除、修复Windows系统，但CMD命令则不行。CMD属于Windows的一部分(接口)，而DOS本身就是一个系统。在Windows操作系统的DOS(CMD命令提示符)，只是系统模拟的DOS操作环境。

CMD命令 和 LINUX命令看起来很相似，因为都是在一个控制台输入一些特定的指令去完成一些特定的操作。其最根本区别是在于Linux SHELL是Linux操作系统的一个用户交互层；而Windows DOS只是Windows系统下的一个小小的应用(终端模拟器)。
