XML

XML( extensible markup language ) 可扩展标记语言，其用来结构化、存储和传输数据。一种很像 HTML 的标记语言，但 XML 标签没有被预定义，需要自行定义标签。

XML 文档，一种树结构，它从”根部”开始，然后扩展到”枝叶”。

实体引用 符号 描述
& lt; < less than
& gt; > greater than
& amp; & ampersand
& apos; ‘ apostrophe
& quot; “ quotation mark

注： 一个实体由三部分构成: 一个和号 (&), 一个实体名称, 以及一个分号 (;)。

XML CSS

通过使用 CSS，可以添加显示信息到 XML 文档中。

XML XSLT

通过使用 XSLT，可以把 XML 文档转换成 HTML 格式。

XML JavaScript

XMLHttpRequest 对象用于在后台与服务器交换数据。 其特点：在不重新加载页面的情况下更新网页在页面已加载后从服务器请求数据、在页面已加载后从服务器接收数据、在后台向服务器发送数据。

XML DOM

定义访问和操作 XML 文档的标准方法。XML DOM 把 XML 文档作为树结构来查看。所有元素可以通过 DOM 树来访问，可以修改或删除它们的内容，并创建新的元素。元素(文本、属性)都被认为是节点。

DTD

DTD ( Document Type Definition )文档类型定义，其用来定义 XML 文档的合法构建模块。通过 DTD，每一个 XML 文件均可携带一个有关其自身格式的描述。应用程序也可使用某个标准的 DTD 来验证从外部接收到的数据。

DTD 可被成行地声明于 XML 文档中，也可作为一个外部引用。

PCDATA

PCDATA (parsed character data) ：被解析的字符数据。文本中的标签会被当作标记来处理，而实体会被展开。

CDATA

CDATA (character data)：字符数据。文本中的标签不会被当作标记来对待，其中的实体也不会被展开。
