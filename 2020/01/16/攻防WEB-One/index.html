0x02 Robots

据题名、题目描述可知，考查robots协议。

Robots协议（也称为爬虫协议、机器人协议等）的全称是“网络爬虫排除标准”（Robots Exclusion Protocol），网站通过Robots协议告诉搜索引擎哪些页面可以抓取，哪些页面不能抓取。

robots.txt 文件是一个文本文件，robots.txt 是搜索引擎中访问网站的时候要查看的第一个文件。robots.txt 文件告诉蜘蛛程序在服务器上什么文件是可以被查看的。

解题：访问 ./robots.txt 文件 >>>得到 f1ag_1s_h3re.php 信息 >>>访问 ./f1ag_1s_h3re.php 文件

0x04 Cookie

据题名、题目描述可知，考查cookie。

cookie 工作原理 ：服务端 向客户端发送的 HTTP 响应中设置 Set-Cookie 消息头 ，Web 服务器可以指定存储一个 cookie。

Set-Cookie:value [ ;expires=date ] [ ;domain=domain] [ ;path=path] [ ;secure]

客户端 向服务端发送的 HTTP 请求中设置 Cookie 消息头， 并且只包含了 cookie 的值，其它的选项全部被去除。

Cookie:user=name

解题：F12查看网络状况 >>>得到 Cookie:look-here=cookie.php >>>访问 ./cookie.php 文件>>> 得到”See the http response”信息

0x09 XFF_REFERER

据题名、题名描述可知，考查X-Forwarded-For、HTTP Referer。

X-Forwarded-For(XFF) 是用来识别通过HTTP代理或负载均衡方式连接到Web服务器的客户端最原始的IP地址的HTTP请求头字段。

X-Forwarded-For: client1, proxy1, proxy2, proxy3

HTTP Referer是header的一部分，当浏览器向web服务器发送请求的时候，一般会带上Referer，告诉服务器该网页是从哪个页面链接过来的，服务器因此可以获得一些信息用于处理。

Referer: URL

解题：X-Forwarded-For: 123.123.123.123 >>> Referer: https://www.google.com

0x11 command_execution

据题名、题名描述可知，考查 linux 命令。

; 每个命令按照从左到右的顺序，顺序执行，彼此之间不关心是否失败，所有命令都会执行。

& 设置此进程为后台进程。技术|Linux 中的 &

| 将前一个命令的结果传递给后一个命令作为输入 。

&& 只有在左边命令返回真，右边命令才会被执行。 只要有一个命令返回假，后面的命令就不会被执行。

|| 只有在左边命令返回假，右边命令才会被执行。 只要有一个命令返回真，后面的命令就不会被执行。

注：Windows：ttl=128 ； Linux ：ttl=64。

解题： 找flag文件：127.0.0.1 && find / -name “flag*”>>>连接flag文件：127.0.0.1 && cat /home/flag.txt 或

|| cat /home/flag.txt 或 & cat /home/flag.txt 或 127.0.0.1 | cat /home/flag.txt
