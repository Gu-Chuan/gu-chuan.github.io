Vim

Jan 27 2020

随笔

Vim 是从 Vi 发展出来的一个文本编辑器。

Vi(Visual interface)没有图形界面的功能强大的编辑器；只能编辑文本内容，不能对字体、段落进行排版；不支持鼠标操作和没有菜单，只有命令。

Vim(Vi improved)vi的提升，支持代码补全、编译、错误跳转等功能。

用法

创建、打开文件：vi 文件名

在终端输入 vi 文件名 。若文件已经存在，则直接打开该文件；若文件不存在，则新建一个文件。

定位打开文件：vi 文件名 + 行数

行数代表定位的行位置，若不指定行数，则直接定位到文件末尾。

文件异常处理

vi/vim 异常退出时，磁盘上可能会保存交换文件。

O键：以只读方式打开；E键：直接编辑；R键：恢复；D键：删除交换文件；Q键：退出；A键：中止。

vi/vim 可分为五种模式，即命令模式(Command mode)、输入模式(Insert mode)、底线命令模式(Last line mode)、可视模式(Visual mode)、替换模式(Replace mode)。

命令模式

用户刚刚启动 vi/vim，便进入了命令模式。在其他模式下按 ESC 键 或 ctrl + [ 快捷键，就回到了命令模式。命令模式是vi/vim 的入口。注：命令模式下每个键都是命令，而非输入字符。

输入模式

在命令模式下按 i、a、o 键，就进入了输入模式。输入模式是正常的编辑文字模式。

底线命令模式

在命令模式下按 : 键，就进入了底线命令模式。底线命令模式是vi/vim 的出口。

可视模式

在命令模式下按 v、V 键，就进入了可视模式。可视模式用于选中文本。

替换模式

在命令模式下按 r、R 键，就进入了替换模式。可视模式用于替换文本。
